Uma Musume: Pretty Derby DB Translate Thai

(อัพเดทข้อมูลม้าใหม่ Smart Falcon) 6/5/2564

โปรแกรมนี้ทำมาเพื่อแปลภาษาเกม Uma Musume: Pretty Derby โดยการแพทส่วนของข้อความตำแหน่ง text_data ในตารางของไฟล์ master.mdb
การสร้างไฟล์แพทนี้ ทำเสร็จสมบูรณ์ทุกอย่างผ่านบราวเซอร์นี้ . เพื่อร่วมให้ข้อมูลการแปลหรือรายละเอียดเพิ่มเติม, คลิกที่ลิ๊งนี้ ใน GitHub Repository.

การใช้งาน

  1. หาไฟล์ทที่ชื่อว่า master.mdb จากนั้นคลิ๊กปุ่มด้านล่าง
    2. โดยปกติแล้วจะอยู่ตำแหน่ง C:\Users\YourUsername\AppData\LocalLow\Cygames\umamusume\master\master.mdb
  2. แทนที่ไฟล์ master.mdb เดิม ด้วยไฟล์ที่สร้างจากเว็บไซด์นี้ : master.mdb แทน

อ่านก่อนโหลด

  • การสร้างไฟล์แพทนี้เป็นการแก้ไขตัวเกมและมันไม่ ได้รับอณุญาติจาก เงื่อนไขการให้บริการ!
    ใช้ม็อดนี้โดยรับความเสี่ยงเอาเอง ! เราจะไม่รับผิดชอบหากไอดีคุณถูกระงับให้บริการ!
  • ไม่ใช่ทุกข้อความที่ถูกแปลลงในไฟล์ master.mdb, ดังนั้นมันไม่ใช่การแปลทั้งเกม ; เช่น. ข้อความสตอรี่ ข้อความการบรรยายการแข่งม้า เป็นต้น.
  • ความยาวของข้อความมันจำกัด ดังนั้น รายละเอียดของสกิล อาจถูกย่อลง หรือตัดให้กระชับขึ้นอาจทำให้งง.
  • ถ้าเกมมันอัพเดทแล้วไฟล์ master.mdb ถูกแก้ไข, ให้กลับมาทำใหม่อีกรอบ.
  • เครื่องมือนี้ทำขึ้นให้ตัวเกม DMM Version (Windows)เป็นหลัก เพราะมันง่ายต่อการติดตั้งไฟล์ม็อด.
    • สามารถเอาไฟล์ master.mdbไปใช้ในเวอร์ชั้นมือถือ Android , Emulator, แต่จำเป็นต้อง Root อุปกรณ์ก่อนถึงจะสามารถนำไฟล์ไปวางทับไฟล์ master.mdb ในตำแหน่ง data/data/jp.co.cygames.umamusume/files/master .
